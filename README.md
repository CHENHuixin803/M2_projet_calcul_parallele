# Projet de Calcul Parallèle : Jeu de la Vie avec MPI

**Cours :** Calcul Parallèle

**Auteurs :** GAO Weihong et CHEN Huixin

## Structure du projet

| Fichier | Description |
| :--- | :--- |
| `row.py` | Implémentation MPI du Jeu de la Vie avec décomposition par **lignes** |
| `col.py` | Implémentation MPI du Jeu de la Vie avec décomposition par **colonnes** |
| `block.py` | Implémentation MPI du Jeu de la Vie avec décomposition en **blocs 2D** |
| `benchmark.py` | Script de banc d'essai pour mesurer les performances (accélération, efficacité, surcoût de communication) sans affichage graphique |
| `Rapport.py` | Rapport final du projet |
